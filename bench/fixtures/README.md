# Bench fixtures

- `small.json` (~1 KB): hand-written config-like document; also the
  target of the typed `parseInto` benchmark.
- `medium.json` (~20 KB) and `large.json` (~300 KB): generated by
  `generate.sh` (deterministic; reruns are byte-identical). The
  generated files are checked in so `zig build bench` needs no
  generation step.
